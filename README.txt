== Description ==
Dit is een administratie tool voor het bedrijf <a href="https://www.specialistinwebsites.nl">Specialist in Websites</a>. De tool zorgt voor een back-end thema van Specialist in Websites, AVG Cookiebot script en meer functies die nog komen!

== Installation ==
### VANAF JE WORDPRESS DASHBOARD
1. Ga naar 'Plugins > Voeg toe'
2. Zoek naar 'SIW TO'
3. Activeer 'SIW TO' vanaf je plugin pagina.
4. Klaar! Je kunt nu de toggles aanzetten en alles instellen.

### VAN WORDPRESS.ORG
1. Download 'SIW TO'.
2. Upload de 'siw-to' folder naar de ‘/wp-content/plugins/’ folder, met jouw favoriete methode (ftp, sftp, scp, etc…)
3. Activeer 'SIW TO' vanaf je plug-in pagina.
4. Klaar! Je kunt nu de toggles aanzetten en alles instellen.

== Frequently Asked Questions ==
= Waarom heb ik deze plug-in nodig? =
Je heb deze plug-in waarschijnlijk niet nodig. Deze plug-in is voornamelijk gemaakt voor medewerkers van Specialist in Websites. De gegeven mogelijkheden van deze plug-in zijn alleen bedoelt voor klanten van SIW.
= Wat is SIW? en TO? =
SIW staat voor <a href="https://www.specialistinwebsites.nl/">Specialist in Websites</a>. TO staat voor "Technisch Onderhoudspakket", wat in principe een luxe woord is voor een All-In-One plug-in met handige foefjes.
